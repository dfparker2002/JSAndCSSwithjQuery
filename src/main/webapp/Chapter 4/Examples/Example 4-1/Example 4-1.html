Setting an attribute value with jQuery's attr() method is really easy, all you have to do is supply the name of the attribute and a value.

The id attribute of this element is set to tmpSetID.

You can also set multiple attributes by supplying an object literal to the attr() method.

The title of this element is set to Hello, World! and the id attribute of this element is set to tmpHelloWorld.

Getting an attribute value is just as intuitive, all you need to do is call the attr() method with the name of the attribute you want to get the value for.

This element's title attribute has a value of .

An attribute can be removed from an element using the removeAttr() method.

This element's class attribute is removed.

Element attributes may also be set via a callback function.
