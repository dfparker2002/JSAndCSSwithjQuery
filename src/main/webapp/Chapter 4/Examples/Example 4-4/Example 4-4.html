jQuery's replaceAll() method takes existing content and replaces it.

jQuery's empty() method removes all children elements (including text) from selected items. jQuery's remove() method completely removes the selected items.

  • replaceAll()
  • replaceWith()
  • empty()
  • remove()
  • clone()

jQuery's clone() method can be used to duplicate elements. Setting its first argument to true also duplicates those elements' event handlers.
